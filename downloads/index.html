Source code

The source code is available at git repository here: https://github.com/simonsalykov/cirqwizard

To get yourself the latest version of source code clone it with the following command:

git clone https://github.com/simonsalykov/cirqwizard.git

The code is provided on GPL v3 license basis.

The easiest way to build sources is to use supplied gradle wrapper:

./gradlew build

and to run the application:

./gradlew run

The recommended Java version is 1.8.0_60. Source code uses Java 8 language features, so it won’t compile on anything lower than 1.8.
