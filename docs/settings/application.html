Settings - Application

Application settings include the following parameters:

Parameter Meaning Default value
Serial port Serial port name (or number) which is used to connect to the machine. If left empty the application will try to auto-detect the port.
Log level Application logging level INFO
Processing threads Number of threads used for tool paths generation. The optimal number is the number of CPU cores in the system (times two for hyper-threading CPUs). If available RAM is not sufficient this number may need to be reduced Number of CPU cores on the computer

Application logs can be found in temporary directory, which is OS dependent.
