Preparation - Homing

After turning the machine on, it needs to find zero points of all axes, which serve as reference points to all coordinates of the machine. This allows a machine to return to the same position after being switched off and on again. This process is referred to as Homing.

To home your machine, after selecting PCB orientation press Home button.

The machine needs to be homed every time you switch it on.
